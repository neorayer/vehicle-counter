Vehicle Counter
==============

This is a coding test for implementation a vehicle counter. Please read REQUESTMENT.md for more detail.

ENV Requirement
----------------
* Maven(>3.0.0)
* JSDK(> 1.8.0)
* JUnit

Build
-----
type:
   
   mvn build
   
Unit Test
---------
Just type:
    
    mvn test
    
Usage
-----
java -cp gov.city.vehicleCounter.App DATA_filePath
